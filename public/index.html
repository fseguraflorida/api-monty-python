Introducción

Bienvenido al entorno de desarrollo. Esta API ha sido diseñada para que los alumnos practiquen peticiones HTTP (GET, POST, PUT, DELETE) sobre estructuras complejas de datos en memoria, totalmente aislados entre sí mediante el uso de tokens.
